\section{Введение}
Исследование математических моделей радиационного теплопереноса \cite{modest_rht}, учитывающих одновременно вклад эффектов теплопроводности и излучения даёт теоретическую основу для инженерных решений в различных областях, таких как производство стекла \cite{OControl_3}, лазерная интерстициальная термотерапия \cite{tse_lasor}, и др. Главной особенностью данных процессов является преобладание компоненты излучения при высоких температурах. Значительное число работ посвящено исследованию задач управления для нестационарных моделей сложного теплообмена \cite{pinnau_identification, pinnau_optimal_control, pinnau_glass}, в которых для описания температурного поля используется нестационарное уравнение теплопроводности, а для моделирования излучения — стационарное диффузионное приближение уравнения переноса излучения. Задача управления для полностью стационарной модели сложного теплообмена рассмотрена в \cite{cheb_origin}.

В работах \cite{covt_last, grenkin_13} задача оптимального управления сводится к bang-bang принципу \cite{grenkin_15}, или аналогичному. Данная работа посвящена нахождению коэфициента отражения участка границы для стационарной модели радиационного теплообмена. В работе доказывается разрешимость полученной экстремальной задачи, приводится алгоритм градиентного спуска и соответсвующие численные эксперименты.

ВТОРОЙ ВАРИАНТ:
Задачи сложного теплообмена являются актуальной областью для исследования в связи с их прикладной значимостью. Подобные задачи интересны как для теоретического исследования \cite{pinnau_optimal_control, greche14, OControl_1} так и для численного моделирования соответсвующих физических процессов \cite{pinnau_glass, kovtanyuk_numerical_simulations,kovtanyuk_an_iterative, pinnau_simplified_pn}. Приведем работы \cite{cheb_origin_8, pinnau_glass}, которые посвящены задачам управления температурой на границе области для нестационарных моделей сложного теплообмена, включающих нестационарное уравнение теплопроводности и стационарное $SP_N$ приближение уравнения переноса излучения. В работе \cite{pinnau_identication_of} рассмотерна задача мултипликативного управления для модели сложного теплообмена.
В работах \cite{covt_last, grenkin_13} задача оптимального управления сводится к bang-bang принципу \cite{grenkin_15}, или аналогичному. 

Близкие к рассмотренной в данной статье, задача управления коэфициентом отражения для полностью стационарной модели исследовалась в \cite{cheb_origin}, для нестационарной модели -- в \cite{grenkin_optimalnoe_upravleine}.
Настоящая работа посвящена нахождению коэфициента отражения участка границы для стационарной модели, по дополнительной информации о температурном поле. Представлено доказательство существования решения соответсвующей экстремальной задачи, а также предложен алгоритм градиентного спуска и соответствующие численные эксперименты.
