\section*{Введение}
Исследование математических моделей сложного теплопереноса \cite{modest_rht}, учитывающих одновременно компоненты теплопроводности и излучения, даёт теоретическую основу для инженерных решений в различных областях,  таких как производство стекла \cite{OControl_3}, лазерная интерстициальная термотерапия \cite{tse_lasor}, и др. Главной особенностью данных процессов является преобладание компоненты излучения при высоких температурах. Значительное число работ посвящено исследованию задач управления для нестационарных моделей сложного теплообмена \cite{pinnau_identification, simlified_control_pinnau, pinnau_glass}, в которых для описания температурного поля используется нестационарное уравнение теплопроводности, а для моделирования излучения — стационарное диффузионное приближение уравнения переноса излучения. Задача управления для полностью стационарной модели сложного теплообмена рассмотрена в \cite{cheb_origin}.

В работах \cite{covt_last, grenkin_13} задача оптимального управления сводится к bang-bang принципу \cite{grenkin_15}, или аналогичному. Данная работа посвящена исследованию задачи управления отражательными свойствами участка границы для стационарной модели сложного теплообмена. В работе доказывается разрешимость задачи управления, выводится система оптимальности и приводятся численные эксперименты по нахождению квазирешения.
